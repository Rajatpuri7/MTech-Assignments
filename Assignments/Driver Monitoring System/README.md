## About This Assignment

This assignment implements a **Driver Monitoring System (DMS)** using computer vision and deep learning techniques to detect driver drowsiness, distraction, and attention state in real-time.

---

## Files

| File | Description |
|---|---|
| `driver_monitor.py` | Main Python script for driver monitoring |
| `DMS.txt` | Assignment notes / documentation |
| `presentation.html` | HTML presentation of the project |
| `requirements.txt` | Python dependencies |
| `Face video_compressed.mp4` | Demo/test video used for the system |

---

## How to Run

```bash
pip install -r requirements.txt
python driver_monitor.py
```
